> HOW TO PLAY

Hmmm, first time playing GTPS on your device? Don't worry! I can tell you how to playing our server on every devices

> Android

  • Virtual Host

    1. Download Virtual Hosts APK from HERE and install it.
    2. DOWNLOAD the file.
    3. Open Virtual Hosts and press on SELECT HOSTS FILE.
    4. Add the hosts file you just downloaded.
    5. Turn on the switch and you're ready to play on our server!

  • PowerTunnel

    1. Download PowerTunnel APK from HERE and install it.
    2. Open the app and open plugins.
    3. Click on the settings icon on HOSTS.
    4. Click on File Hosts DNS.
    5. Paste on YOUR_PTUNNEL_URL.
    6. Now go back and click on Connect and you're ready to play on our server!

  • HostGo

    1. Install Hosts Go APK from HERE and install it.
    2. Open Hosts Go and press on Host Editor.
    3. Press + and then Add IP Address and Domain.
    4. Now go back and click on START and you're ready to play on our server!

  • IOS

    1. Download and install Surge 5 from App Store.
    2. Open Surge 5 and press SETUP button at the top right corner of the screen.
    3. Press ALLOW and enter your device password.
    4. Press DEFAULT on top left.
    5. Find Edit in Text Mode button and press it.
    6. Paste the following code below.
    7. Press Done at the top right corner.

    9. You're ready to play on our server!

  • Windows

    1. Run NOTEPAD as administrator.
    2. Press File and then Open....
    3. Navigate to C:\Windows\System32\drivers\etc.
    4. Change file type from Text Documents (*.txt) into All files (.).
    5. Double click on hosts.
    6. Remove everything before inserting anything described here.
    7. Paste the host.
    8. Save the file and you're ready to play on our server.

  • MacOS & Linux

    1. Open terminal as root user.
    2. Type sudo nano /etc/hosts.
    3. If command is not found, install nano from your package repositories.
    4. Remove everything else from the file and paste the host.
    5. Press CTRL + X and press y.
    6. You're ready to play on our server.
